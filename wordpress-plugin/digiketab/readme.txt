کتابخانه دیجیتال کامل با قابلیت مطالعه آفلاین کتاب‌ها، خواننده PDF داخلی، و پشتیبانی کامل از زبان فارسی.

== Description ==

**دیجی‌کتاب** یک پلاگین کتابخانه دیجیتال کامل برای وردپرس است که امکان ایجاد یک کتابخانه آنلاین حرفه‌ای را فراهم می‌کند.

### ویژگی‌های اصلی:

📚 **مدیریت کتاب‌ها**
- افزودن، ویرایش و حذف کتاب‌ها
- آپلود فایل‌های PDF
- دسته‌بندی و برچسب‌گذاری
- مدیریت نویسندگان

📖 **خواننده داخلی**
- مطالعه آنلاین کتاب‌ها
- خواننده PDF پیشرفته
- کنترل‌های صفحه‌ورق
- تنظیمات فونت و پوسته

🔍 **جستجو و فیلتر**
- جستجوی متنی پیشرفته
- فیلتر بر اساس دسته‌بندی
- فیلتر بر اساس نویسنده
- مرتب‌سازی نتایج

📱 **طراحی ریسپانسیو**
- سازگار با موبایل و تبلت
- نمای قفسه‌ای و شبکه‌ای
- طراحی فارسی راست‌چین

📤 **اشتراک‌گذاری**
- لینک مستقیم کتاب‌ها
- QR کد برای دسترسی سریع
- اشتراک در شبکه‌های اجتماعی

⚙️ **پنل مدیریت**
- داشبورد آماری
- گزارش‌گیری
- تنظیمات جامع
- پشتیبان‌گیری

### کد کوتاه (Shortcode):

```
[digiketab_library]
```

### پارامترهای اختیاری:

- `category="نام-دسته"` - نمایش کتاب‌های یک دسته خاص
- `limit="12"` - محدود کردن تعداد کتاب‌ها
- `view="grid"` - نوع نمایش (grid یا shelf)

### مثال:
```
[digiketab_library category="ادبیات" limit="8" view="shelf"]
```

== Installation ==

### نصب خودکار:
1. به پنل مدیریت وردپرس بروید
2. روی "افزونه‌ها" > "افزودن" کلیک کنید
3. "دیجی‌کتاب" را جستجو کنید
4. پلاگین را نصب و فعال کنید

### نصب دستی:
1. فایل ZIP پلاگین را دانلود کنید
2. در پنل مدیریت به "افزونه‌ها" > "افزودن" بروید
3. روی "آپلود افزونه" کلیک کنید
4. فایل ZIP را انتخاب و آپلود کنید
5. پلاگین را فعال کنید

### پس از نصب:
1. از منوی "دیجی‌کتاب" در پنل مدیریت دیدن کنید
2. کتاب‌های نمونه اضافه می‌شوند
3. از کد کوتاه `[digiketab_library]` در صفحات استفاده کنید

== Frequently Asked Questions ==

= چگونه کتاب اضافه کنم؟ =

1. به "دیجی‌کتاب" > "افزودن کتاب" بروید
2. عنوان، توضیحات و تصویر کتاب را وارد کنید
3. فایل PDF کتاب را آپلود کنید
4. دسته‌بندی و نویسنده را تعیین کنید
5. کتاب را منتشر کنید

= چگونه کتابخانه را در صفحه نمایش دهم؟ =

کد کوتاه `[digiketab_library]` را در هر صفحه یا پست قرار دهید.

= آیا از زبان فارسی پشتیبانی می‌کند؟ =

بله، پلاگین کاملاً فارسی است و از راست‌چین نویسی پشتیبانی می‌کند.

= چه فرمت‌هایی برای کتاب پشتیبانی می‌شود؟ =

در حال حاضر فرمت PDF پشتیبانی می‌شود. در نسخه‌های آینده فرمت‌های بیشتری اضافه خواهد شد.

= چگونه طراحی کتابخانه را تغییر دهم؟ =

می‌توانید از طریق فایل CSS سفارشی یا در بخش "تنظیمات" پلاگین، ظاهر را تغییر دهید.

= آیا امکان پشتیبان‌گیری وجود دارد؟ =

بله، از بخش "تنظیمات" می‌توانید از تمام کتاب‌ها پشتیبان‌گیری کنید.

== Developer Notes ==

### هوک‌ها و فیلترها:

```php
// فیلتر برای تغییر تعداد کتاب‌ها در هر صفحه
add_filter('digiketab_books_per_page', function($count) {
    return 16; // تغییر از 12 به 16
});

// اکشن پس از افزودن کتاب
add_action('digiketab_book_added', function($book_id) {
    // کد سفارشی
});

// فیلتر برای تغییر استایل‌ها
add_filter('digiketab_css_classes', function($classes) {
    $classes[] = 'my-custom-class';
    return $classes;
});
```
